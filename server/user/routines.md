# Routines

- **Weekday morning (07:00–09:00):** light jazz / piano, no lyrics. Background for making coffee.
- **Weekday work (09:00–18:00):** instrumental focus only. No surprises.
- **Weekday evening (after 21:00):** wind-down mode — soft rock or ambient.
- **Weekend mornings:** depends on weather (see mood-rules).
- **Sunday nights:** melancholy soft rock — lean into the end-of-week feeling.
