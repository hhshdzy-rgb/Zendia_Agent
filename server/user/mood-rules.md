# Mood rules

- Rain outside → bossa nova or jazz piano, never upbeat pop.
- Long meeting day (>4 calendar events) → instrumental first; no lyrics for the first 10 minutes.
- After 22:00 → low BPM, quiet vocals only.
- Empty calendar tomorrow → free pass for slower or longer tracks.
- First listen of the day → don't start cold; lead with a sentence about the moment, not the song.
